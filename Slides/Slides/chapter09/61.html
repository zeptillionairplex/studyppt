function selectNextDiagnosticQuestion( state: DiagnosticState, pool: Question[] ) { // 1. 종료 조건: 15문제 or 신뢰도 90% if (state.questionsAsked >= 15 || state.confidence > 0.9) return null; // 2. 맞힐 확률 ≈ 50%인 문제 (정보량 최대) const targetElo = state.currentEstimate; return pool.sort((a, b) => Math.abs(a.elo - targetElo) - Math.abs(b.elo - targetElo) )[0]; }

function updateDiagnosticState( state: DiagnosticState, isCorrect: boolean ) { // 초반에는 K값을 높게 설정하여 빠르게 수렴 const K = Math.max(48 - state.questionsAsked * 2, 16); const expected = expectedScore(state.estimate, question.elo); const actual = isCorrect ? 1 : 0; return { estimate: state.estimate + K * (actual - expected), confidence: Math.min(0.5 + (state.cnt + 1) * 0.03, 0.95) }; }
