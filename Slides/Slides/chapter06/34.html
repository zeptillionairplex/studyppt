2-TIER CACHE STRATEGY
1계층: 단원 기반 (Broad) Student Use

hash(단원ID + 유형 + 난이도)
→ 같은 단원이면 선생님 달라도 재활용 (Free Tier)

2계층: 문서 버전 기반 (Fine) Teacher Use

hash(버전ID + 태그ID + 유형)
→ 내가 만든 원문, 내가 태깅한 포인트 정확 반영
