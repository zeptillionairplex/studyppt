프롬프트 캐싱 적용 코드

단 한 줄의 코드로 AI 비용을 획기적으로 절감하는 방법

Implementation Pattern
Anthropic (Claude) Implementation
TS
const response = await anthropic.messages.create({ model: "claude-3-5-sonnet-20240620", max_tokens: 1024, system: [ { type: "text", text: SYSTEM_PROMPT, // 3,500 tokens of instruction cache_control: { type: "ephemeral" } // ← 이 한 줄로 90% 할인 } ], messages: [ { role: "user", content: userMessage } ] });
OpenAI (GPT-4o) Implementation
TS
const response = await openai.chat.completions.create({ model: "gpt-4o", messages: [ { role: "system", content: SYSTEM_PROMPT }, // 자동 캐싱 적용 (코드 변경 없음) { role: "user", content: userMessage } ], // 1,024 토큰 이상의 동일 프롬프트 접두사가 반복되면 // OpenAI가 자동으로 감지하여 50% 할인 적용 });

핵심 포인트

Anthropic은 명시적 파라미터가 필요하지만 할인율이 더 높고(90%), OpenAI는 완전 자동이지만 할인율이 절반(50%)입니다. 두 경우 모두 코드 복잡도 증가 없이 즉시 적용 가능합니다.
