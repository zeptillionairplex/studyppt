외부 AI Agent REST API

AI 에이전트가 콘텐츠를 생산하고 선생님이 검수하는 자동화 파이프라인

Base URL: https://api.educontent.com/v1
Agent Workflow
01

인증 & 권한 확인

Bearer Token 기반 API Key 인증으로 접근 권한을 확인

Header: Authorization
02

가이드 학습

태깅 가이드와 OpenAPI 명세를 읽고 작성 규칙 파악

GET /docs/tagging-guide
03

콘텐츠 작성 (Agent)

AI Agent가 규칙에 맞춰 원문 작성 및 태깅 수행

Internal Agent Process
04

Import 업로드

작성된 콘텐츠를 JSON/Markdown 형태로 업로드 (자동 태깅 옵션)

POST /documents/import
05

선생님 검수

업로드된 문서는 'Draft' 상태로 생성되어 선생님 검수 대기

Notification Trigger
06

승인 및 활용

검수 완료 후 정식 문서로 변환, 문제 생성 및 배포 가능

Status: Active
