Chapter 7. 문제 검수 워크플로

AI가 만든 문제를 바로 학생에게
줄 수 없는 이유

"AI는 초안을 만들고, 최종 품질은 교사가 보장한다.
이것이 Human-in-the-Loop의 핵심입니다."

AI가 잘하는 것 (Assistant)
  • 원문에서 핵심 용어를 정확히 찾아 빈칸으로 생성
  • 정답과 해설을 원문 기반으로 빠르게 작성
  • 복잡한 JSON 포맷을 실수 없이 출력
  • 대량의 문제(10~100개)를 수 초 내에 생성
AI가 실수할 수 있는 것 (Risks)
  • 오답 선택지가 너무 쉽거나 터무니없는 경우
  • 문맥상 정답이 2개 이상 가능한 애매한 문제
  • 한국어 조사/어미 처리 오류 ("을/를" 부자연스러움)
  • 원문에 없는 내용을 지어내는 할루시네이션
AI = 빠르고 성실한 조교 (초안 생성)
선생님 = 최종 결재권자 (검수 및 승인)
